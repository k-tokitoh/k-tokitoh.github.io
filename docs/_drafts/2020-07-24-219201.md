# 知ったこと

  * SOAPは型つきでリソースベース、RESTに先行
  * クエリするときフィールド名にエイリアスをつけられる
  * 「インラインフラグメント」なるほどな
  * フラグメントはクライアント側の話であって、スキーマ（やサーバー側）の話ではない
  * 共通のフィールドがあるならinterfaceにすべき、ないならunionにすべき。
